# Data Storage

이 디렉토리는 프로젝트에서 사용하는 모든 파일 기반 데이터를 저장하는 공간입니다. 데이터의 처리 단계(Stage)에 따라 하위 디렉토리를 구분합니다.

## 디렉토리 구조

### `raw/` (Bronze Layer)
*   외부 소스에서 수집된 **원본 그대로의 데이터**입니다.
*   이곳의 데이터는 읽기 전용(Immutable)으로 취급하는 것이 원칙입니다.
*   예: API 응답 JSON, 수집된 CSV 원본.

### `processed/` (Silver Layer)
*   Raw Data를 정제, 구조화, 필터링한 **가공된 데이터**입니다.
*   분석이나 머신러닝 모델 학습, 애플리케이션 서비스에 직접 사용될 수 있는 품질을 가집니다.
*   예: 결측치가 처리된 CSV, 표준화된 데이터셋.

### `dart/` (Legacy)
*   이전 구조에서 사용되던 디렉토리입니다. 점진적으로 `raw/dart/` 또는 `processed/dart/`로 마이그레이션 예정입니다.
